Como Funciona?

O código VIN sempre deve obedecer essa lei de formação, logo terá sempre 17 caracteres. (As letras I, O, Q não são utilizadas, por sua similaridade com os números 1, 0 (zero)) Apesar de possuir diversos caracteres e uma combinação quase infinita, decifrar o VIN não é tão complicado. Basta familiarizar-se com o significado das letras e números no código que o compõe.
